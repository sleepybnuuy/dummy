# Dummy

a Dalamud plugin to reinstate my favorite little guy.

created [by mistake](https://bsky.app/profile/bwuny.heliosphe.re/post/3llnmsctmn22o) and missing-in-action [since 2025](https://na.finalfantasyxiv.com/lodestone/topics/detail/6f824223a7e10da7b9b7dfc84f626d10d4df88b3/), you can now witness the rebirth of a legend at your local Kugane Aetheryte plaza! the striking dummy created by this plugin is non-interactible and purely for cosmetics and laughs.
